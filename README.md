# Shadow Star Trading Website

## Project Overview
Shadow Star Trading Website is a promotional landing page developed for an intraday trading advisory platform. The website provides information about trading services, share market learning courses, and links to social media platforms including Telegram and Instagram.

The website was designed to introduce the trading service, promote share market courses, and allow users to join the trading community.

---

## Features
- Responsive landing page design
- Navigation menu with multiple sections
- Share market course information
- Social media integration (Telegram and Instagram)
- Popup notification for joining Telegram group
- Contact information and location details
- Registration form integration
